# Actividad

Crea una página similar pero para la venta de libros. Cada libro tendrá como información:

- id
- Título
- ISBN
- Escritores
- Precio incial
- Precio en oferta
- Imagen del libro
- puntuación entre 1 y 5
- Opiniones totales
- Descripción

Implementa como mínimo:

- Carrito de la compra
- Busqueda por nombre o palabra de la descripción
